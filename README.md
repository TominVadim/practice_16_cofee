# Практическая работа 16. Заказ кофе

## Описание
Система оформления заказа кофе с использованием паттерна Builder.

## Правила заказа
- **Обязательно**: основа (base) и размер (size)
- **Основа**: espresso, americano, latte, cappuccino
- **Размер**: small, medium, large
- **Молоко**: none, whole, skim, oat, soy
- **Сиропы**: максимум 4 разных
- **Сахар**: от 0 до 5 ложек
- **Лед**: +0.2 рубля

## Запуск тестов
```bash
python3 main.py
```
## Технические требования

- Python 3.10+
- Стандартная библиотека
- Один файл main.py
